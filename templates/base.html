HOW TO SEARCH

Narrow down your search using filters album, artist, track, year, upc, tag:hipster, tag:new, isrc, and genre

Example:

techno, album:new
will return techno albums released within 2 weeks

The artist and year filters can be used while searching albums, artists and tracks. You can filter on a single year or a range (e.g. 1955-1960). The album filter can be used while searching albums and tracks. The genre filter can be used while searching artists and tracks. The isrc and track filters can be used while searching tracks. The upc, tag:new and tag:hipster filters can only be used while searching albums. The tag:new filter will return albums released in the past two weeks tag:hipster can be used to return only albums with the lowest 10% popularity.
